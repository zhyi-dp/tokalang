Toka currently builds from source to ensure you have the latest performance optimizations and LLVM integration.

## Prerequisites

Before installing Toka, ensure your system has the following dependencies:

- **LLVM 20**: The Toka compiler is built on the LLVM 20 framework.
- **Clang 20**: Required for linking the generated LLVM IR into final executables.
- **CMake**: Version 3.15 or higher.
- **Zsh**: Recommended for the best development experience on macOS.

## Building the Compiler

1. Clone the repository:
   ```bash
   git clone https://github.com/zhyi-dp/tokalang.git
   cd tokalang
   ```

2. Create a build directory and run CMake:
   ```bash
   mkdir build && cd build
   mkdir src
   cmake ..
   make
   ```

3. The compiler binary `tokac` will be located in `build/`.

## The Toka Build Tool

The `toka` tool handles project management and incremental builds. It is a Toka application itself!

1. Build the tool from source:
   ```bash
   ./tokac tools/toka/src/main.tk > build/toka.ll
   clang build/toka.ll -o build/toka -isysroot $(xcrun --show-sdk-path)
   ```

2. Add the tool to your `PATH`:
   ```bash
   export PATH="$PATH:$(pwd)/build"
   ```

## Environment Configuration

For the best experience, add these to your `.zshrc` or `.bashrc`:

```bash
# Path to the compiler and build tool
export PATH="$PATH:/path/to/tokalang/build"

# Critical: Path to the standard library
export TOKA_LIB_PATH="/path/to/tokalang/lib"

# The Clang compiler to use for linking
export TOKA_CLANG="/usr/local/bin/clang"
```

> [!IMPORTANT]
> Ensure that `TOKA_CLANG` points to a version of Clang that matches the LLVM version of your `tokac` (currently LLVM 20).
