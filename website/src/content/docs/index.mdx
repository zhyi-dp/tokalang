## Key Innovations

<CardGrid stagger>
	<Card title="Absolute Safety Without Lifetime Annotations" icon="shield">
		Pioneering the Single-Hat Principle and PAL Checker. Eliminates dangling pointers and data races at compile time with zero runtime overhead. Enjoy 100% memory safety without endlessly fighting the compiler.
	</Card>
	<Card title="Zero-Cost Abstraction & C Interop" icon="rocket">
		Built on LLVM 20 with No Garbage Collection (No GC). Its memory layout closely mirrors C/C++, allowing seamless zero-overhead inline calls to the vast existing C ecosystem.
	</Card>
	<Card title="Transparent Concurrency & Error Flow" icon="puzzle">
		No implicit black magic schedulers. Standard functions and `async` functions have clear physical boundaries. Paired with the native `!` error propagation operator, banish callback hell and `if err != nil` waterfalls.
	</Card>
	<Card title="Blazing Fast Modern Toolchain" icon="setting">
		Forget messy CMake build scripts. Toka features the built-in `toka run` and an AI-Native Package Manager. A simple `toka.json` fetches and builds global dependencies from the edge network.
	</Card>
</CardGrid>
