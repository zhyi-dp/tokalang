## Key Innovations

<CardGrid stagger>
	<Card title="AOT Native Compilation" icon="rocket">
		Experience lightning-fast execution via `toka run`. Features transparent auto-linking and LLVM optimization levels (-O0 to -O3) for maximum performance.
	</Card>
	<Card title="Safe Native Memory" icon="shield">
		Ownership is baked into the grammar with **^T** (Unique) and **~T** (Shared) pointers. Enjoy zero-cost abstractions and hardened standard containers.
	</Card>
	<Card title="Attribute Token System" icon="pencil">
		Visual chemical formulas for types. Combine **name#** (Mutable) and **Type?** (Nullable) for precise control without keyword bloat.
	</Card>
	<Card title="Soul-Identity Duality" icon="document">
		Philosophically split between **Soul** (Value) and **Identity** (Address). Eliminate ambiguity between reference assignment and content mutation.
	</Card>
</CardGrid>
