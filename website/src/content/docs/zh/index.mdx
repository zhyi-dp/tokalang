## 核心创新

<CardGrid stagger>
	<Card title="无生命周期标注的绝对安全" icon="shield">
		首创单帽原则 (Single-Hat Principle) 与 PAL 检查器。在编译期以零运行时开销杜绝悬空指针与数据竞争。享受 100% 的内存安全，免受与编译器搏斗的折磨。
	</Card>
	<Card title="零成本抽象与 C 原生互操作" icon="rocket">
		底层基于 LLVM 20 构建，无垃圾回收 (No GC)。内存布局与 C/C++ 极其贴近，支持无缝内联调用现有的庞大 C 语言历史遗产（如 SQLite、OpenGL）。
	</Card>
	<Card title="透明并发与原生错误流" icon="puzzle">
		拒绝隐式的“黑魔法调度器”。普通函数与 `async` (异步挂起) 具有清晰的物理边界。配合原生的 `!` 错误传播符，彻底告别回调地狱和 `if err != nil` 瀑布。
	</Card>
	<Card title="极速的现代化工具链" icon="setting">
		忘记繁琐的 CMake 构建脚本。内置 `toka run` 与 AI 原生包管理器。只需一个简单的 `toka.json`，即可在全局网络中极速拉取并构建全部依赖。
	</Card>
</CardGrid>
