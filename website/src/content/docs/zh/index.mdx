<CardGrid stagger>
	<Card title="语法级 RAII" icon="rocket">

	</Card>
	<Card title="属性令牌系统" icon="pencil">

	</Card>
	<Card title="灵肉合一 (Soul-Identity)" icon="document">
		在哲学上区分 **Soul** (值) 与 **Identity** (地址)。消除引用赋值与内容修改之间的歧义。
	</Card>
