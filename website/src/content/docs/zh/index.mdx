## 核心创新

<CardGrid stagger>
	<Card title="AOT 原生编译" icon="rocket">
		通过 `toka run` 体验闪电般的执行速度。支持透明自动链接与 LLVM 优化级别 (-O0 到 -O3)，榨干硬件性能。
	</Card>
	<Card title="安全原生内存" icon="shield">
		所有权机制被植入语法层面。基于 **^T** (唯一) 和 **~T** (共享) 指针，享受零成本抽象与经过严苛测试的标准容器。
	</Card>
	<Card title="属性令牌系统" icon="pencil">
		类型的“化学分子式”。组合使用 **#** (灵魂可写) 和 **?** (实体可空) 等短令牌实现精确控制，告别臃肿的关键字。
	</Card>
	<Card title="灵肉合一 (Soul-Identity)" icon="document">
		在哲学上区分 **Soul** (值) 与 **Identity** (地址)。消除引用赋值与内容修改之间的歧义。
	</Card>
