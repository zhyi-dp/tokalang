title: 元组与数组
description: Toka 语言中的固定大小和动态集合。
---

Toka 提供了对集合的内置支持，从基础的固定大小数组和元组，到更灵活的切片和动态向量。

## 元组 (Tuples)

元组是固定大小、异质的值集合。它们使用圆括号定义。

```toka
auto t = (10, "Hello", 3.14)
auto (val, msg, p) = t // 解构
```

## 数组 (Arrays)

数组是同质的值集合。可以使用方括号直接创建固定大小的静态数组。

```toka
auto arr = [1, 2, 3, 4, 5] // 固定 5 个元素大小
```

## 一等动态数组与切片 (Dynamic Arrays & Slices)

在 Toka 中，`[T]` 代表一等动态数组连续内存序列。根据附加的指针形态符号，它能作为**胖指针** (Fat Pointer，内部包含指针与运行时长度) 表达不同的内存模型：

- `^[T]` (独占数组): 动态分配的数组，严格拥有数据所有权，并使用运行时长度进行依次安全析构。
- `&[T]` (引用切片): 对序列的借用视图，不拥有数据。
- `~[T]` (共享数组): 基于引用计数的动态数组切片块。

```toka
auto ^slice# = new [4]i32(0) // 动态分配包含 4 个元素的数组

pub fn sum(&data: [i32]) -> i32 { // 切片借用传递，无所有权转移
    auto res# = 0
    for x in data {
        res# += x
    }
    return res
}
```

## 特殊语法：for-or

Toka 的 `for` 循环有一个独特的 `or` 子句，在集合为空时执行。

```toka
for x in data {
    import std/io::println
    println("项: {}", x)
} or {
    println("集合为空！")
}
```

## 标准库：Vec

虽然数组是固定大小的，但标准库通过 `std/vec` 提供了 `Vec<T>` 用于动态扩容。

```toka
import std/vec::Vec

pub fn main() -> i32 {
    auto v# = Vec<i32>::new()
    v.push(10)
    v.push(20)
    return 0
}
```
