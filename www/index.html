Innovation

Key Innovations

⚓

Syntactic RAII

Ownership is baked into the grammar. ^T (Unique) and ~T (Shared) pointers make resource management invisible and zero-cost.

💎

Soul-Identity Duality

Philosophically split between Soul (Value) and Identity (Address). Eliminate ambiguity between reference assignment and content mutation.

⚛️

Orthogonal Attributes

Visual chemical formulas for types. Combine # (Mutable), ? (Nullable), and ! (Both) for precise control without keyword bloat.

🛡️

@encap Security

Explicit encapsulation. Control field visibility and resource lifecycles at the structural level with mandatory impl @encap blocks.

🔄

Contract-Based Flow

Everything is an expression. Unique for-or syntax ensures loop results are guaranteed even on empty iterations.

⚡

LLVM Powered

Native performance with LLVM 20. Advanced optimizations for coroutines, opaque pointers, and strict memory safety checks.

Attribute Token System

The language of memory, visualized.

Token Value Meaning (Soul) Identity Meaning (Address)
# Writable: Mutable fields Reseatable: Can point to new data
? Nullable: Value can be none Nullable: Address can be nullptr
! Writable & Nullable Reseatable & Nullable
^ - Unique Owner (RAII)
~ - Shared Owner (RC)
& Borrow: Temporary view Reference: Stable view

Write with Elegance

main.tk 
shape Rect (w: i32, h: i32)

impl Rect@Shape {
    fn area(self) -> i32 {
        return self.w * self.h
    }
}

fn main() {
    auto ^r = new Rect(w = 10, h = 20) // Unique Pointer
    
    auto a = r.area()
    println("The area is {}", a)
}
